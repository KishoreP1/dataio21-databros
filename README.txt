# Data/IO 2021 Hackathon Project

## [Driven Insights-(https://driveninsights.org/)

### What is it
Driven insights is a website that allows for easy exploration of the V2X driving dataset. It won the Honda Data Exploration Challege as well as best presentation at Data/IO.

### Technologies used
The Holoviz ecosystem of libraries was used to make the interactive visualization, primarily Panel, Pandas and Altair. A bootstrap website hosted on Netifly was used to deploy our project.
